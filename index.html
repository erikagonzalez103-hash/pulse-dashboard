Overview

The Pitch

The 30-second pitch is the team's shared starting point. Memorize the four anchor phrases — the rest fills itself in.

Every city has a pulse — energy that builds around places in real time.

And nobody's measuring it.

Yelp captures reputation. Instagram captures attention. Google captures intent. None of them tell you what's actually happening right now.

That's Pulse.

We're the momentum layer for cities — surfacing what's accelerating, neighborhood by neighborhood, in real time.

Reputation is static. Momentum is dynamic.

And with the World Cup landing in Dallas this summer, momentum is the whole game.

Brand

Visual Language

Elegance with edge — Roman inscription weight in the wordmark, near-black depth as the canvas, gold as the accent that signals heat, and a deep cool red as the brand's pulse. Sharper edges than typical app design; brighter lights than typical luxury.

Primary Mark
Compass mark with cardinal directions. Use at all sizes; primary identity for app icons, social, and standalone applications.
Heartbeat Mark
Heartbeat variant. Use for animated contexts, motion, and moments emphasizing real-time pulse — e.g., loading states, live event surfaces.
Wordmark
PULSE
Cinzel 600 weight, +0.18em tracking, gold on dark only. Use as text identity in headers, signatures, and any context already containing the mark.

Color Usage

Gold — primary brand accent. Wordmark, key actions, anchor highlights. Use sparingly to preserve weight.
Pulse Red — signals heat, urgency, live moments. Heat-scale fire end, live indicators, alerts. Never as background.
Cream — body text on dark. Default reading color.
Stone — secondary text, metadata, non-critical UI labels.

For the Team

Using This Hub

This page is the project's front door. The actual editing of documents happens elsewhere — and that's by design. The hub stays clean and orienting; collaboration happens where collaboration tools are good.

01
Read here, edit there
Use this hub to orient. For live editing and comments, put the docx files in Google Drive or Notion. Update the download links in this page to point to the live versions.
02
Deploy in five minutes
Drop this folder onto Netlify Drop or push to a GitHub Pages repo. Both are free and produce a shareable URL the team can bookmark. Files in the same folder become downloadable links automatically.
03
Update as it grows
Add new artifacts as cards in the relevant section (Research, Strategy, Investors). Keep the metadata strip honest — page count and last-updated date matter more than they look like they do.
04
Suggested team setup
Hub on Netlify (orientation). Docs in Google Drive (editing). Tasks in Linear or a simple Notion board (work). Comms in Slack. The hub is the front door; the rest is the house.
